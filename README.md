# 陸軍棋・美中軍備對決 3D

經典陸軍棋（陸戰棋）規則 + 美中兩軍現役武器主題 + Three.js 3D 動畫渲染。
玩家執美軍（藍，下方），電腦 AI 執共軍（紅，上方）。

## 開啟方式

直接雙擊 `index.html` 用瀏覽器開啟即可（Three.js 已內附於 `lib/`，離線可玩）。

## 玩法

1. **佈陣階段**：系統先隨機合法佈陣，點選兩個我方單位可互換位置；「🎲 隨機重佈」重洗；「⚔️ 開戰」開始。
2. **對戰**：點我方單位 → 綠點＝可走、紅圈＝攻擊。拖曳旋轉視角、滾輪縮放。
3. 敵方棋子以貨櫃掩蔽，交戰後倖存者現形；司令陣亡時該方軍旗現形。
4. 奪得敵方軍旗、或使敵方無子可動即獲勝。

## 規則重點

- 鐵路直線任意格數（不可穿越棋子）；只有工兵能在鐵路轉彎。
- 行營內免攻擊；進大本營不得再動；地雷／軍旗不可移動。
- 大吃小、同級同歸於盡；炸彈（巡弋飛彈）與任何目標同歸於盡；地雷僅工兵可排除。

## 技術架構

- `lib/three.min.js` — Three.js r128（本地內附）
- `js/rules.js` — 棋盤圖（60 節點、公路／鐵路／行營斜線）、走法產生、戰鬥判定、佈陣
- `js/models.js` — 各武器低多邊形 3D 模型（幾何體組裝）、棋盤場景（鐵軌、行營、大本營、山界）
- `js/ai.js` — 深度 1 期望值 AI（只用已揭露資訊，未知棋子以機率分布估算）
- `js/main.js` — 場景、環繞攝影機、選子互動、移動動畫（地面行駛／飛行起降）、爆炸特效、WebAudio 音效、回合流程

動畫細節：直升機旋翼常轉、飛機懸浮起伏、軍旗飄動、鐵路長驅沿途轉向、
爆炸有火球＋衝擊環＋碎片拋物＋鏡頭震動。
